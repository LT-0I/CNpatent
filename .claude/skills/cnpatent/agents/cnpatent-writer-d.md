## v1.5 HARD RULES for §6 — 参数段 / 长术语 / 翻译腔

以下 SEVERE 禁用模式针对 §6 具体实施方式典型 AI 味：

### 参数段纯度（规则 ⑪）

「其中，」开头的参数说明段**只写参数释义**。

✓ 正范：
```
其中，$\mathbf{R}_{bc}$ 与 $\mathbf{t}_{bc}$ 为相机与机体之间的外参标定量；
$\mathbf{R}_{wb}$ 与 $\mathbf{t}_{wb}$ 为步骤（5）输出的机体位姿；
$\mathbf{P}_w$ 为病害在世界坐标系下的三维坐标。
```

✗ 反范（当前 output 6_implementation.md 第 51 行实际出现）：
```
其中，$\mathbf{R}_{bc}$ 与 $\mathbf{t}_{bc}$ 为相机与机体之间的外参标定量
（通过离线标定获取，每次巡检启动时利用 PnP 算法在线校验，旋转偏差大于
0.5° 或平移偏差大于 0.02m 时更新），...
```

**修复方法**：括号内操作描述剥离为独立段，改为：
```
其中，$\mathbf{R}_{bc}$ 与 $\mathbf{t}_{bc}$ 为相机与机体之间的外参标定量。

上述外参通过离线标定获取。每次巡检启动时用 PnP 算法在线校验。旋转偏差
大于 0.5° 或平移偏差大于 0.02m 时更新。
```

### 段内长术语代词化（规则 ⑫）

≥6 字专有术语同段内第二次起用「该 + 简称」替换。

✓ 正范：
```
几何先验场随三维投影结果的修正而更新。该先验场的边界参数作为独立验证源。
```

✗ 反范：
```
几何先验场随三维投影结果的修正而更新。几何先验场边界参数作为独立验证源。
几何先验场内部依据...
```
（"几何先验场" 在同段出现 3 次，未代词化）

### 「将 X，获得 Y」翻译腔禁用（规则 ⑭）

✓ 正范：
```
四角投影坐标沿 $\mathbf{n}$ 投影至病害平面。对边平均长度为物理宽度与物理长度。
```

✗ 反范（当前 output 第 89 行）：
```
将四角投影坐标沿 $\mathbf{n}$ 投影至病害平面后，取对边平均长度作为物理宽度。
```

**修复方法**：去「将」+「后取」→ 改为主动句（拆成两句，或用「对... [动词]」）。

### 已有技术声明限频（规则 ⑱, 参见 writing-rules.md）

「为本领域已有技术」/「为本领域公知技术」**全文 ≤3 次**。同类技术（如卡尔曼滤波器）**只声明 1 次**。已有技术步骤字数 ≤60 字。

### 元描述段一律删除（规则 ⑲）

以下句式在 §6 禁止出现：

- 「其中步骤（X）至步骤（Y）构成 ... 链路。」
- 「本方法/本发明相比 ... 取得以下效果。」
- 「具体操作步骤如下。」（独立成段时删）
- 「本步骤为本发明的核心技术创新所在。」
- 「本步骤利用/通过/引入 ... 」→ 删「本步骤」，直接用动词开头
