# cnpatent v1.4 + v1.5 Writer / Reviewer 硬约束（共享单点真相）

> 所有 writer (A/B/C/D) 和 reviewer 在写作/审查时**必须**遵循本表的 14 条硬约束。
> 这些约束来自《专利文档"AI味"特征分析报告》（v1 17 类 + v2 16 类）对先前生成
> 稿的 AI 味命中，由 cnpatent-humanizer v1.5 的 audit 自动检测（`--enable-all`）；
> 违反即红灯。

## 14 条硬约束

| # | 约束 | 反例 | 正例 |
|---|---|---|---|
| 1 | **单句 ≤ 40 字**（句号/问号/叹号之间） | "本发明的目的是提供一种XXX方法，以解决YYY问题，并进一步解决ZZZ问题。"（150 字） | 切两句：拆问题句 + 拆方案句 |
| 2 | **单段 ≤ 200 字** | P85 单段 520 字塞进问题+方案+效果+触发+退化 | 按论点拆 3 段 |
| 3 | **段内分号 ≤ 2 个** | "X；Y；Z；W；V" | 用句号拆段 |
| 4 | **编号分点总起句 < 10 字**（用户明确要求） | "（4）病害目标检测与几何先验驱动的误检抑制"（17 字） | "（4）病害检测误检抑制"（8 字） |
| 5 | **禁止 8 类发明八股开篇/收尾** | "本发明的目的是提供..." / "本发明的优势体现在：" / "本发明提供...相比现有技术具有..." / "本方案的必要性..." / "本方案的收敛性..." / "本方案的效果..." / "本步骤为本发明的核心技术创新所在" / "综上所述，本发明..." | 直接陈述方法 / 直接按条列出优势 / 删除元话语 |
| 6 | **禁止风格标签动词名词化** | 锚定、降维、注入、兜底、自举闭环、同根、落于 | 依据、简化、施加、回退、相互校正、同源、属于 |
| 7 | **禁止强调副词修饰技术名词** | "持续追踪""固有的过修正""全局残余偏差""整体虚警率""有机组成部分" | 删除副词：追踪 / 过修正 / 残余偏差 / 虚警率 / 组成部分 |
| 8 | **同一三元并列（A、B、C）跨段 ≤ 2 次** | "稀疏、延迟、含噪" 在全文 5 段重现 | 第 3 次起改散文："观测稀疏且延迟，并含噪"或分拆成单独论点 |
| 9 | **复合长名词短语拆短**（≥4 个名词无虚词即拆） | "激光惯性紧耦合位姿估计与位姿质量指标输出" | "紧耦合融合激光与惯性估计位姿，输出位姿质量指标" |
| 10 | **被动书面化改主动** | "对状态向量施加分布式反向修正"；"由阻尼比例予以限制"；"以阻尼注入方式对向量施加修正" | "对状态向量做分布式反向修正"；"阻尼比例限制"；"用阻尼对向量做修正" |
| 11 | **参数段纯度**（v1.5 新增） | `其中，R_bc 与 t_bc 为外参（通过离线标定获取，每次巡检启动时利用 PnP 算法在线校验，偏差大于 0.5° 时更新）...` | 参数段：`其中，R_bc 与 t_bc 为相机与机体之间的外参。`<br>操作段：`上述外参通过离线标定获取。每次巡检启动时用 PnP 算法在线校验。` |
| 12 | **段内长术语代词化**（v1.5 新增） | 同段出现 `几何先验场` ×4 | 第二次起替换为 `该先验场 / 先验场`；跨段仍用全称（术语锁定）|
| 13 | **话题切换必分段**（v1.5 新增） | 单段含 `架构上...观测处理方面...稳定性方面` 三层 | 每个标志词处断段；或用 `（1）（2）（3）` 编号替代 |
| 14 | **「将 X，获得 Y」翻译腔禁用**（v1.5 新增） | `将四角投影坐标沿 n 投影至病害平面后，取对边平均长度作为物理宽度` | `四角投影坐标沿 n 投影至病害平面。对边平均长度为物理宽度。` 或 `对四角投影坐标做投影，得到...` |

### §4b vs §6 步骤呈现差异（v1.5 明确）

| 节 | 标题呈现 | 单步字数 | 范例 |
|---|---|---|---|
| §4b 技术解决方案 | 标题与正文**合并**为一段 `（N）标题：正文` | ≤150 字 | `（1）数据采集与同步：可见光相机、激光雷达与 IMU 同步采集...` |
| §6 具体实施方式 | 标题**独立**成行 + 正文另起 + 子步骤 a）b）c） | 200-400 字（创新步骤可长） | `（1）时域累积扫描的主轴方向估计\n\n如图1所示，...\n\na）...\nb）...` |

## 自检时机

- **Writer 写完每个子节后**：逐条对照上表，当场改，不等 Reviewer
- **Reviewer 审查时**：Rubric-C 扩展项，任一条违反 → `fix_spec` 退回 Writer
- **Phase 3 步骤 0 humanizer 闸门**：`audit.py --enable-all`，score ≥ 25 退回 Reviewer

## 例外

| 场景 | 例外规则 |
|---|---|
| 六节 numbered 子步骤标题 | H18 <10 字规则仍生效；但子步骤内句子字数可到 60 字（技术细节复杂） |
| 公式段 | 不计单句字数（`$...$` 视为原子） |
| 法律声明段 | 全部 10 条约束豁免 |
| 术语锁定表中的术语 | 复合名词拆短规则豁免该术语本身 |
