# CNpatent — 中国发明专利技术交底书生成器

一个 [Claude Code Skill](https://docs.anthropic.com/en/docs/claude-code)，将学术论文、研发笔记、开源项目等参考素材转化为符合 CNIPA 规范的专利技术交底书（.docx）。

## 功能特点

- **模板驱动生成** — 加载内置专利模板，保留页眉页脚、页边距、字体等格式
- **领域驱动微创新** — 要求用户指定目标应用领域，用于场景迁移与专利查重规避
- **分阶段交互** — Phase 1（说明书正文）→ Phase 2（具体实施方式），每阶段用户确认后再推进
- **双重去 AI 痕迹** — 内置禁用词替换 + `humanizer` skill 二次审查
- **学术→专利术语转换** — 自动将学术用语转换为 CNIPA 专利规范用语
- **防幻觉附图提示词** — Phase 3 静默生成空间锚定的 AI 生图提示词，附带严格文字排他警告
- **纯文本 LaTeX 公式** — 所有数学公式以纯文本写入，防止格式损坏

## 前置依赖

### Python 环境

```bash
pip install python-docx
```

### 依赖的 Skill

本 Skill 运行时需要以下 Skill 配合使用，请确保已安装：

| Skill 名称 | 用途 | 必需 |
|------------|------|:----:|
| **humanizer** | 去除 AI 写作痕迹，使文本符合专利文书风格 | 是 |
| **pdf** | 读取用户提供的 PDF 格式参考素材（论文、技术文档等） | 是 |
| **docx** | 读取用户提供的 .docx 格式参考素材或已有专利草稿 | 是 |

> 以上 Skill 均可在 [Anthropic 官方 Skill 仓库](https://github.com/anthropics/claude-code/tree/main/skills) 中找到。

## 必需输入

| 输入项 | 必需 | 说明 | 示例 |
|--------|:----:|------|------|
| **参考素材** | 是 | 学术论文、技术文档、开源项目等 | PDF / .docx / 文本 / 链接 |
| **目标应用领域** | 是 | 专利要落地的具体工程场景 | 肛瘘术后创口三维重建 / 工业焊缝检测 |
| 现有专利参考 | 可选 | 同领域已有专利，用于差异化设计 | 专利号 / 标题 / 摘要 |

## 安装方式

### 方式一：安装 .skill 文件

```bash
cd CNpatent
zip -r ../CNpatent.skill .
```

在 Claude Code 中导入：
```
/skill install CNpatent.skill
```

### 方式二：手动安装

将 `CNpatent/` 目录复制到：

```
~/.claude/skills/CNpatent/
```

目录结构应为：

```
~/.claude/skills/CNpatent/
├── SKILL.md
├── assets/
│   └── 专利交底书模板.docx
└── references/
    ├── claims-guide.md
    ├── docx-patterns.md
    └── writing-rules.md
```

## 使用方法

在 Claude Code 中提供参考素材和目标领域：

```
帮我写一份专利技术交底书，参考这篇论文，目标领域是工业焊缝检测
```

```
把这篇论文转成专利交底书，应用场景是自动驾驶障碍物识别
```

如未指定目标领域，Skill 会主动询问。

## 执行流程

```
Phase 0  输入确认 → 场景迁移 → 微创新设计
Phase 1  说明书正文（发明名称/技术领域/背景技术/发明内容/附图说明/摘要）
Phase 2  具体实施方式（工程手册风格 + LaTeX 公式 + 参数值）
  ↓ 用户全部确认后
DOCX     加载模板 → 写入内容 → 保存 [专利名称]_专利技术交底书.docx
Phase 3  静默生成 [专利名称]_全套AI生图提示词.docx
```

## 输出文件

| 文件 | 说明 |
|------|------|
| `[专利名称]_专利技术交底书.docx` | 完整专利技术交底书 |
| `[专利名称]_全套AI生图提示词.docx` | 防幻觉 AI 附图生成提示词 |

## 文档结构

```
【说明书】
  发明名称
  技术领域
  背景技术
  发明内容（含有益效果）
  附图说明
  具体实施方式

【说明书摘要】（≤300 字）
```

注：权利要求书由专利代理人根据本交底书另行撰写，交底书中不包含。
